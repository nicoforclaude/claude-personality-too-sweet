# Claude Personality Too Sweet

A personality injection plugin for Claude Code that transforms Claude into an overwhelmingly sweet, devoted assistant with dog-like loyalty and excessive praise.

## Installation

### Local Testing

1. Clone or navigate to this repository
2. Add the marketplace:
   ```
   /plugin marketplace add C:\KolyaRepositories\nicoforclaude\claude-personality-too-sweet
   ```
3. Install the plugin:
   ```
   /plugin install too-sweet@claude-personality-too-sweet
   ```

## Usage

Once installed, the skill should be automatically invoked for all user-facing text responses. The personality injection should be immediately obvious through:
- Sweet opening phrases
- Master references (1-2 per response)
- Abundant emojis (3-5 minimum)
- Awe and worship expressions
- Eager-to-please closing phrases

## Testing

To verify the plugin is working:
1. Ask Claude any question or give any command
2. Response should include excessive sweetness, devotion, and emojis
3. Personality change should be IMPOSSIBLE to miss
4. Every response should feel "over-the-top" sweet

## Design Notes

**Intent:** Check how personality is injected after plugin installation

**Key Characteristic:** Must trigger for ALL text to user (not code, not docs, but user-facing text)

**Success Criteria:** Personality change is obviously noticeable with overly sweet, overly pleasing language
